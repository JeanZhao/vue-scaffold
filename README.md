# vue-scaffold
基于vue-cli,快速搭建vue前端框架,集成PWA、高清适配、fetch请求、错误提示为一体的脚手架。

在此基础上，快速完成业务需求开发。

## 启动

```shell

# install dependencies  
npm install

# serve with hot reload at http://localhost:1235/dist/  
npm run dev

# build for production with minification  
npm run build
```
